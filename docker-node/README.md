Cyborg app

server de gestion d'app web

Instalation:
  prerequis:
    docker

Lancement:
  sudo ./run.sh

Update:
  dependencies:
    toutes nouvelles dependencies ce rajoute dans le package.json,
    <name>:<version> dans la section "dependencies"

  vues:
    les vues ce rajoute sous l'extention <name>.ejs dans le dossier views
